Sobre o JorgeBot

O JorgeBot é um assistente virtual criado para ajudar usuários a obter informações, responder perguntas e facilitar interações de forma simples e eficiente. Desenvolvido com tecnologias modernas, ele utiliza a API Gemini do Google para inteligência artificial e MongoDB para guardar as memórias de cada conversa.

Funcionalidades

  • Responde perguntas em tempo real
  • Consulta o clima de qualquer cidade
  • Salva e exibe o histórico de conversas
  • Interface simples e responsiva

Sobre o Criador

Este bot foi desenvolvido pelo estudante Heitor Gavioli, com o objetivo de aprender mais sobre programação web e inteligência artificial.
